Worth to know:

Before using the plugin, you should know some of its behavior/bugs that may be the reason for not using this library at this time. Problems will be solved as a way is found and the author has free time. If there is a problem in this section, then it has not been solved yet :)

Offset:

At set speeds of less than 5000, and even with some iterations of the tape, it can be shifted both to the right and to the left when calculating the path.

Partially fixed.

Also:

If you know how to solve the problems above, or you find another bug, you can:

  • Create a pull request with changes.
  • Write about a bug or suggestion in issues.
